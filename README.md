# Design System UI Library

A reusable component-based UI system built with JavaScript and Tailwind CSS.

## Features
- Reusable components (Button, Input, Card)
- Multiple variants and states
- Design tokens for consistency
- Responsive styling using Tailwind
- Scalable architecture

## Tech Stack
- HTML
- JavaScript (Vanilla)
- Tailwind CSS
- Vite

## Components
- Button (primary, secondary, danger)
- Input field
- Card layout

## Purpose
Built to demonstrate UI consistency, scalability, and reusable frontend architecture.
